The Very Basics



What is the World Wide Web?

The World Wide Web is comprised of a collection of HTML Documents. When a person accesses a website through a web browser, the computer sends and HTTP request through a server, and retrieves the correct HTML Document, which a web browser is then able to interpret and display. This Video is a good explanation.


HTML

- Stands for Hypertext Markup Language.

- Is the language predominantly used for content on the web.

- Consists of text content (what you see) & markup (what it looks like

- This video is a good refresher.


Elements & Tags

- HTML markup consists of tags. There are serparate opening and closing tags, with the content (text) between the tags being affected.

- Opening tags generally look like < b > , while closing tags look almost identical, but with an added backslack < / b >

- The tags & content together are known as an element. HTML documents consist of many elements.

- Some tags do not need a closing tag, these are called void tags.

- Here is a video that explains further.


Inline & Block Elements

- Elements are either inline or block

- Block elements form an invisible box around the text.

- So far inline elements are void tags, while block elements are not.
